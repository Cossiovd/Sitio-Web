Nombre del proyecto: Clínica Veterinaria - Sitio web

Descripción: desarrollo de un sitio web moderno donde se ofrecen diferentes servicios para mascotas. Se incluye secciones como inicio, servicios, contacto, información de la clinica y elementos visuales atractivos que ayudan a tener una buena experiencia del usuario. 

Tecnologias utilizadas: HTML5 y CSS

Url del proyecto: https://cossiovd.github.io/Sitio-Web/

Instrucciones de instalación o visualización: 
1. Clonar el repositorio: ```git clone https://github.com/Cossiovd/Sitio-Web.git```
2. Entrar al proyecto: ```cd Sitio-Web```
3. Abrir el archivo principal index.html o usar la extensión Live Server en VS Code
